# Publishing Form

The following components are visual steps of our publishing form.

## How to use ?

You can move from step to step using the sidebar. The button at the end of each step will not move you to the next one in Storybook.

## Usage

The steps are ready to use with all the front-end validation logic. Renderless Provider components and composables are also provided to handle requests to udata api.
