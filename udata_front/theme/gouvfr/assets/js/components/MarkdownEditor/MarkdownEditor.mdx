# Markdown Editor

A full fledged Markdown editor with WYSIWYG for ease of use and GFM for added capabilities.

It allows :
- basic text formatting (titles, **bold** and *italic*)
- table and table management (add / move / align / remove row, add / move / align / remove column)
- adding link on text
- adding image
- lists
- code block
- quote
