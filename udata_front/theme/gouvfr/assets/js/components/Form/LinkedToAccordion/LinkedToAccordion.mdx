# Linked To Accordion

This component allows an input (or any other element) to open an accordion component on focus.


<Canvas of={LinkedToAccordion.LinkedToClosedAccordion} />

## Accessibility

You must provide the accordion `id` as `aria-describedby` for users of assistive technologies.
They will read the content, even if the accordion is closed.
