# Range Picker

Used to pick dates for temporal coverage selection.

Supports getting passed a value in the `value` prop (useful to pass value from URL), will call onChange fn on change.

Value is a string composed of the two dates forming a range as such : `YYYY-MM-DD-YYYY-MM-DD`
